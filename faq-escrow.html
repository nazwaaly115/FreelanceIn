    FAQ Rekening Bersama (Escrow)

    Apa itu Rekening Bersama FreelanceIn?

    Dana proyek UMKM disimpan sementara di escrow platform hingga pekerjaan disetujui. Mahasiswa baru menerima pembayaran setelah UMKM melepaskan dana.

    Bagaimana alur pembayarannya?

    1. UMKM menerima proposal mahasiswa
    2. UMKM deposit dana via Midtrans Snap (sandbox/production)
    3. Mahasiswa mengerjakan proyek
    4. UMKM review deliverable dan lepaskan dana

    Apakah aman untuk UMKM?

    Ya. Dana tidak langsung ke mahasiswa. Jika pekerjaan tidak sesuai, UMKM dapat minta revisi sebelum melepaskan dana.

    Integrasi Midtrans

    Platform mendukung Midtrans Snap sandbox. Setel MIDTRANS_SERVER_KEY di file .env untuk aktivasi gateway nyata.
